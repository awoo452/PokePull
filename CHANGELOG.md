# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/
spec/v2.0.0.html).

## [0.0.2] - 2026-03-27

### Changed
- Renamed the app folder and package name to PokePull.
- Updated README branding to PokePull and fixed formatting.

## [0.0.1] - 2026-03-27

### Added
- Initialized the Pokemon Pulse Next.js app.
- Built the one-button UI for fetching random Pokemon.
- Added a server-side API proxy to the Pokemon API.
- Documented local development and Amplify deployment.
- Added an Original 151 toggle, retryable error state, and a recent pull history.
- Added loading skeletons to smooth the fetch experience.
